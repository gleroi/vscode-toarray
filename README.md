# toarray 

toarray converts hexadecimal, decimal strings to array of hexadecimal numbers.

## Features


- 00010203 becomes 0x00, 0x01, 0x02, 0x03
- 12,2,3,255 becomes 0x0C, 0x02, 0x03, 0xFF
